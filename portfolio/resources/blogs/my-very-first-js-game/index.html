My Very First JS Game

Jan 15, 2023 | Ed Garcia

Learning programming is fun, but creating a game is more fulfilling! Just a month before posting this blog, I was just another user or a gamer. I never thought that I could create a simple but fun web game made only with pure JavaScript.

Uplift Code Camp, a bootcamp institution in the Philippines that helps a no-programming-knowledge like me to learn programming one syntax at a time.

I still remember the day that I've read a post from Uplift regarding a scholarship. This scholarship will grant you a ticket to be part of their new batch, a batch of aspiring website developers.

Luckily, I was one of the ten who got the ticket. I was very happy, but I was like, "programming, please be easy on me."

The journey of learning how to program is twice as hard as I initially imagined. There were terms and words that new to my ears. Not to mention the syntax that was not making sense to me.

Honestly, I was having a hard time coping up at first, but the programmer-spirit in me is stronger than my laziness. LOL!

After a ton of discussions about syntax, arrays, objects, and functions, finally, I was able to understand how programming really works.

Programming is not a rocket-science (as I thought of) afterall.

Here's one of the fruits of my labor, I created a simple but fun game just for you. Made with nothing but HTML, CSS, and pure JavaScript.

I hope you enjoy it. Launch the game here Unzooom!
